Data Formatting, Data Cleaning, Speed Calculation

After reading in the datasets, I got rid of the columns I wasnt interested in, leaving the GPS data, heartrate, and distance. I then formatted the time and elliminated any rows with missing data.

wo1 <- wo1RAW

#deleting unnecessary columns in wo1
wo1$Sport <- NULL; wo1$..Sport <- NULL; wo1$ns1.TriggerMethod <- NULL; wo1$ns1.Name <- NULL; wo1$ns1.TotalTimeSeconds <- NULL
wo1$ns1. <- NULL; wo1$ns1.DistanceMeters <- NULL; wo1$ns1.UnitId <- NULL; wo1$ns1.ProductID <- NULL
wo1$ns1.Calories <- NULL; wo1$ns1.Id <- NULL; wo1$StartTime <- NULL; wo1$ns1.Intensity <- NULL

#renaming columns in wo1
names(wo1)[1] <- "DateTime"; names(wo1)[2] <- "Latitude"; names(wo1)[3] <- "Longitude"; names(wo1)[4] <- "Altitude"
names(wo1)[5] <- "Distance"; names(wo1)[6] <- "HR"

#Formatting time
wo1$DateTime <- sub(pattern="T", replace=" ", wo1$DateTime)
wo1$DateTime <- sub(pattern=".00.*", replace="", wo1$DateTime)
wo1$DateTime <- as.POSIXct(wo1$DateTime, format = "%Y-%m-%d %H:%M:%S")
wo1$Date <- sub(pattern=" .*", replace="", wo1$DateTime)
wo1$Date <- as.Date(wo1$Date)
wo1$Time <- sub(pattern=".* ", replace="", wo1$DateTime)

#Eliminating rows with any missing data
wo1 <- wo1[complete.cases(wo1),]

I calculated speed using the distance and time variables. I then got rid of any junk data, such as negative speeds and speeds inconsistent with the workout I was doing (AKA: speeds over 15mph).

#Adding Speed
wo1$Speed <- rep(NA, length(wo1$DateTime))
for(i in 2:length(wo1$DateTime)){
  wo1$Speed[i] <- (wo1$Distance[i]-wo1$Distance[i-1])*0.0006213712/(as.numeric((wo1$DateTime[i]-wo1$DateTime[i-1]))/3600)
}

#Cleaning up dataframe
wo1 <- wo1[wo1$Speed>0,]        #Gets rid of rows with negative speed (not possible)
wo1 <- wo1[wo1$Speed<15,]        #Gets rid of rows with unrealistic speed
wo1 <- wo1[2:length(wo1$DateTime),]

Regression Analysis

Finally, I performed a regression analysis on each of the workouts HR vs Speed. I also compared log(HR) vs Speed and HR vs log(Speed), but straight HR vs Speed produced the highest R-squared values for each of the workouts.

Workout 1

ggplot(data=wo1, aes(x=Speed, y=HR))+geom_point()+geom_smooth(method="lm", color="blue")+xlab("Speed [mph]")+ylab("HR [BPM]")+ggtitle("Workout 1 - HR vs Speed")

Further Work

I would investigate ways to replace any missing data. While this is not necessary for the analysis, it would improve the readibility of the plots that include latitude and longitude.

Additionally, I plan on recording data for more speeds, not just ~4mph and ~7.5mph. Ideally, I would start out at 4mph and slowly increase my speed to ~8mph. This would give a better understanding of how HR truly differs with speed. Id be curious to see if the relationship would be linear or if it would be better fit by something polynomial.
